Workshop 2014

Clayton Miller will be leading a half-day workshop “Python for Building Modellers” at the 2014 ASHRAE–IBPSA-USA Building Simulation Conference in Atlanta, GA, USA in the morning of Monday, Sept. 8, 2014.

Participants should download the Canopy Express of Canopy Academic versions of Python before the session

The agenda is as follows:

Agenda: Morning Session – 8:00 – 9:50AM – Python Basics and Pre-processing

  • Introduction to Python
  • Installing, running a script
  • Data types
  • Flow control
  • Functions and Modules
  • Exercise: Open and modify a text file

Afternoon Session – 10:10AM-12:00PM – Post-Processing and Time Series Data

  • Data Analysis Packages
  • Exercise: Simulation Calibration Process
  • Exercise: Data Visualization
  • The Future: Parallel Computing, Resources for Q&A, etc
  • Further resources, conclusions

How to join the workshop: 

Participants can register directly through the ASHRAE conference website in the pre-conference workshops section

For any questions, contact Clayton at miller.clayton@arch.ethz.ch
